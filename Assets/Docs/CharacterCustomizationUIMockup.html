Thumbnail Policy

icon이 있으면 icon 사용, 없으면 generated thumbnail, 그것도 없으면 prefab preview/text fallback.

Toast: Equipped: TShirt 012
1. Data 확장 Part definition에 icon/thumbnail 필드를 추가하고, 없을 때 prefab preview로 fallback합니다.
2. Thumbnail 생성 EditorWindow에서 Generate Thumbnail 버튼으로 PNG를 만들고 part icon에 연결합니다.
3. Runtime UI Category tab, part grid, selected detail, equip/unequip 버튼을 플레이용 canvas에 구현합니다.
4. Ontology 적용 EquipPart 호출 후 renderer, world facts, capability rules, toast, save/load를 연결 검증합니다.
